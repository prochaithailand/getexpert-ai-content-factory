# Sprint 14 Comparative User Feedback Questionnaire

This survey questionnaire gathers feedback from testers regarding how the no-Streamlit-logo VPS interface impacts user confidence, branding trust, and overall checkout operations.

---

## Comparative Survey Questionnaire

### Section 1: การลบเครื่องหมายการค้าและการสร้างแบรนด์ (Branding & Trust)

#### Q1: ความน่าเชื่อถือของแบรนด์เมื่อลบโลโก้ Streamlit
*   *คำถาม:* การลบแถบ Deploy สีดำด้านบน และคำว่า "Made with Streamlit" ด้านล่างออกไป ทำให้คุณมีความเชื่อมั่นในความเป็นเว็บโปรแกรมแบบ SaaS มืออาชีพของ GetExpert AI เพิ่มขึ้นเพียงใด?
*   *ตัวเลือก:*
    1.  เพิ่มขึ้นอย่างมาก (ดูเป็นระบบสากลของบริษัทเอง)
    2.  เพิ่มขึ้นเล็กน้อย
    3.  ใกล้เคียงของเดิม / ไม่มีความแตกต่าง

#### Q2: ความรู้สึกในการตัดสินใจชำระเงินค่าเครดิต (Purchase Confidence)
*   *คำถาม:* หน้าตาแอปที่สะอาดสะอ้านและเป็นสัดส่วน (ไม่มีสัญลักษณ์ Streamlit) ช่วยเพิ่มความมั่นใจในการจ่ายเงินค่าโอนเครดิตผ่านช่องทางแอดมินทาง LINE OA หรือไม่?
*   *ตัวเลือก:*
    1.  ช่วยเพิ่มความมั่นใจในการโอนเงินได้มากขึ้นอย่างชัดเจน
    2.  ไม่มีผลต่อการตัดสินใจโอนเงิน

### Section 2: ประสบการณ์แสดงผลบนมือถือ (Mobile UI Cleanliness)

#### Q3: ความเป็นระเบียบของหน้าเว็บเมื่อเปิดบนมือถือ
*   *คำถาม:* การจัดระเบียบองค์ประกอบใหม่บนหน้าจอมือถือ (โดยเฉพาะ Layout บล็อกแจ้งโอนเงิน และระบบประวัติตระกูลการสร้างคอนเทนต์ย้อนหลัง) ช่วยให้อ่านและคัดลอกข้อมูลได้ง่ายขึ้นหรือไม่?
*   *คำตอบ:* คะแนน 1 (ยังกวนใจอยู่/จัดวางไม่ดี) ถึง 5 (สะอาดตาและจัดระเบียบดีมาก)

#### Q4: ความคิดเห็นเพิ่มเติมเกี่ยวกับการปรับปรุงหน้าบ้าน (Feedback)
*   *คำถาม:* หากต้องแนะนำจุดที่ควรแก้ไขในตัว UI หน้าบ้านเวอร์ชันสะอาดนี้เป็นลำดับถัดไป คุณต้องการให้ปรับปรุงส่วนใดมากที่สุด? (ปลายเปิด)
