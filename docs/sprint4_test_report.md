* **สิ่งที่ทดสอบ:** กรอกฟอร์มพร้อมข้อมูล Topic, Keyword และบริบท Target Audience = "เจ้าของร้านอาหาร SMEs", Business Type = "ร้านชาบูบุฟเฟต์", Content Goal = "เพิ่มการแชร์เพจ", Tone = "เป็นกันเอง สไตล์นักกิน"
* **ผลลัพธ์ที่คาดหวัง:**
  - ข้อมูลใน Google Sheet คิวงานถูกสร้างในตำแหน่งแถวถัดไป
  - คอลัมน์ป้อนเข้าใหม่ U ถึง X มีข้อมูลจัดเก็บตามที่กรอกลงฟอร์มจริง
* **ผลการทดสอบจริง:** **ผ่านการทดสอบ (PASSED)**

### 2. ทดสอบความเข้ากันได้ย้อนหลัง (Backwards Compatibility)
* **สิ่งที่ทดสอบ:** ในตารางชีต มีแถวข้อมูลเก่าของระบบ Sprint 2-3 อยู่ด้วย (ข้อมูลความยาวเพียง 20 คอลัมน์ A:T) จากนั้นสั่งรัน Engine `python main.py`
* **ผลลัพธ์ที่คาดหวัง:**
  - ตัวประมวลผลอ่านแถวเก่าได้สำเร็จ ปลอดภัยจากอาการดัชนีคอลัมน์เกินขอบเขต (Index out of range)
  - แถวเก่าถูกดึงไปเขียนบทความ Gemini และสร้างบล็อกร่าง Blogger อัปเดตกลับชีตได้ปกติโดยไม่มีอาการ Error
* **ผลการทดสอบจริง:** **ผ่านการทดสอบ (PASSED)**
